Parses command-line arguments based on predefined argument definitions.

This function processes command-line arguments provided to the program and matches them against a set of predefined argument definitions. It supports string, boolean, and integer argument types. If the -h or --help flag is provided, it prints the usage information.

Parameters
argcThe number of command-line arguments.
argvThe array of command-line argument strings.
argDefCThe number of predefined argument definitions.
argDefsThe array of predefined argument definitions of type IndraArg.
Returns
true if the arguments were successfully parsed, false otherwise.
